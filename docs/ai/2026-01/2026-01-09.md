# API接口文档：发布公众号文章

## 接口说明
将文章发布到指定的微信公众号草稿箱

## 接口地址
POST https://wx.limyai.com/api/openapi/wechat-publish

## 请求头
X-API-Key: your-api-key
Content-Type: application/json

## 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| wechatAppid | string | 是 | 公众号AppID（从"获取公众号列表"接口获取） |
| title | string | 是 | 文章标题（最多64字符） |
| content | string | 是 | 文章内容（支持Markdown或HTML） |
| summary | string | 否 | 文章摘要（最多120字符） |
| coverImage | string | 否 | 封面图URL |
| author | string | 否 | 作者名称 |
| contentFormat | string | 否 | 内容格式：markdown（默认）或html |
| articleType | string | 否 | 文章类型：news（普通文章，默认）或newspic（图文/小绿书） |

## 请求示例 (cURL)
```bash
curl -X POST https://wx.limyai.com/api/openapi/wechat-publish \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "wechatAppid": "wx1234567890",
    "title": "测试文章",
    "content": "# 标题\n\n这是文章内容...",
    "summary": "这是文章摘要",
    "contentFormat": "markdown"
  }'
```

## 响应示例（成功）
```json
{
  "success": true,
  "data": {
    "publicationId": "uuid-here",
    "materialId": "uuid-here",
    "mediaId": "wechat-media-id",
    "status": "published",
    "message": "文章已成功发布到公众号草稿箱"
  }
}
```

## 响应示例（失败）
```json
{
  "success": false,
  "error": "公众号不存在或未授权",
  "code": "ACCOUNT_NOT_FOUND"
}
```

## 错误码说明
| 错误码 | 说明 |
|--------|------|
| API_KEY_MISSING | API密钥未提供 |
| API_KEY_INVALID | API密钥无效 |
| ACCOUNT_NOT_FOUND | 公众号不存在或未授权 |
| ACCOUNT_TOKEN_EXPIRED | 公众号授权已过期 |
| INVALID_PARAMETER | 参数错误 |
| WECHAT_API_ERROR | 微信接口调用失败 |
| INTERNAL_ERROR | 服务器内部错误 |

## 注意事项
1. 文章内容支持Markdown和HTML格式
2. 图片会自动上传到微信服务器
3. 发布后文章会出现在公众号的草稿箱中
4. 如果未指定封面图，会自动从内容中提取第一张图片作为封面

## 小绿书（图文模式）说明
当 `articleType` 设置为 `newspic` 时，发布为图文消息（小绿书）：
- 系统会自动从内容中提取所有图片（最多20张）
- 内容中必须包含至少1张图片
- 文字内容仅支持纯文本，最多1000字
- 不支持HTML标签和复杂格式
- 图片会自动上传到微信服务器
- 适合以图片为主的内容发布

## 发布示例（小绿书）
```bash
curl -X POST https://wx.limyai.com/api/openapi/wechat-publish \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "wechatAppid": "wx1234567890",
    "title": "小绿书测试",
    "content": "这是纯文本描述\n\n![图片1](https://example.com/image1.jpg)\n![图片2](https://example.com/image2.jpg)",
    "articleType": "newspic"
  }'
```
