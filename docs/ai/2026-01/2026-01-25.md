**Qwen3-TTS** 是阿里云 Qwen 团队开发的**开源文本转语音（TTS）模型系列**，主要功能包括：

1. **语音合成**：支持 10 种语言（中英日韩德法俄葡西意）的高质量语音生成
    
2. **核心功能**：
    
    - **自定义语音**：预设多种音色（Vivian、Serena 等 9 种）
    - **语音设计**：通过自然语言描述创建目标声音（如"撒娇稚嫩的萝莉女声"）
    - **语音克隆**：仅需 3 秒参考音频即可克隆声音
3. **技术特点**：
    
    - 超低延迟流式生成（97ms 端到端合成延迟）
    - 支持指令控制情感、语调、语速
    - 端到端架构，避免传统 LM+DiT 的信息瓶颈
4. **应用场景**：实时语音助手、有声内容创作、配音、多语言语音交互等
    

GitHub Star 数已达到 **3.7k**，项目已开源并提供 Python 包 `qwen-tts` 和 Web UI 演示。
