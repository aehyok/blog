最近，发现身边好多龙虾养殖户还不知道使用OpenClaw内置命令。

着实让这只龙虾有点憋屈。

要用好龙虾，常用命令还是要了解一下：

——💬 Session (会话管理)——

◽️/new 或 /reset

开启新会话 / 重置上下文。清空当前的对话记忆（对模型而言是重新开始），避免之前的对话干扰或占用过多的 Token。

◽️/compact

压缩上下文。不完全清空，而是将之前的长对话总结成精简内容，释放空间以便继续长线任务。

◽️/stop

紧急停止。强制中断模型当前正在生成的回复、正在运行的脚本或后台卡住的工具。

◽️/clear

清屏。主要用于清理聊天界面的历史消息记录（UI 层面）。

◽️/focus

专注/绑定模式。在支持线程的平台（如 Discord），用于将当前聊天流独占绑定给特定的智能体或任务。

——🧠 Model (模型与参数控制)——

◽️/model

切换模型。随时无缝切换当前会话使用的底层模型（如 /model gpt-4o 或 /model gemini）。

◽️/think

调整思考深度。控制模型在回答前的内部思考量（可选 off/minimal/low/medium/high/xhigh）。级别越高，逻辑越严密但耗时越长。

◽️/verbose <on|off|full>

调试/啰嗦模式。开启后，原本在后台默默执行的工具调用细节、错误堆栈和内部指令都会打印出来，适合排查问题。

◽️/fast <status|on|off>

极速模式。开启后，机器人会禁用大部分复杂的工具调用（如查网页、跑代码），强制变为“纯文本聊天”模式，回复速度最快。

——🛠️ Tools (系统工具与状态)——

◽️/commands 或 /help

查看命令。显示当前系统支持的所有快捷指令菜单

◽️/status

查看状态。输出当前机器人的运行仪表盘，包括底层环境、模型配额剩余、耗时等信息。

◽️/export

导出记录。将当前的对话历史（包含系统隐藏的 Prompt）导出为 Markdown 或 HTML 文件保存到本地。

◽️/usage

用量统计开关。控制是否在每次回复的末尾附加本次消耗的 Token 数量和计费成本。

◽️/skill

指定技能。强制让机器人调用工作区内安装的某个特定技能脚本（比如 /skill weather）。

——🤖 Agents (子智能体/多任务管理)——

◽️/agents

查看智能体。列出当前正在为你工作的线程级 Agent 列表。

◽️/kill <id|all>

强杀子任务。当你派生出后台子 Agent 去写代码或查资料，但发现它卡死或跑偏时，用这个直接终止它。

◽️/steer

实时纠偏。如果后台的子 Agent 正在运行，你可以用这个命令中途给它下达新指令（比如：“换个方向，去查一下 2024 年的数据”），而不需要杀掉重来。

其实，要是忘了，输入“/help”就会拉出可用的命令列表。




玩OpenClaw 养龙虾的小伙伴们看过来，这些常用的内置命令，能让你在养虾的归途上，更加游刃有余，还能省不少token。在聊天对话框中直接发送/命令即可使用，欢迎点赞收藏备用。
