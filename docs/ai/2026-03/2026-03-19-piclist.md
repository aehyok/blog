PicList+ Cloudflare R2 好像是绝美的夫妻搭档一样，再配合上Obsidian + Claude Code 简直就是完美的第二大脑配置组合。

PicList管理图片工具完全开源免费，Mac、Window、Linux三端都可使用。
而且支持中英切换，非常友好。而且功能丰富，不仅仅是上传，还可以管理云端的文件资源管理器，搜索 复制、删除、上传 下载通通都有。


而Cloudflare R2 如果是个人使用可以说是完全免费。
每月10G的免费月存储，假设5M一张图片，大致可以免费上传2000张图片。一篇文章10张图片，一个月可以写200篇了，足够了吧。 出站流量也就是我们文章访问图片的次数是每个月1000万次之后才会收费。 而且即使收费，也很便宜，详情请看图片。

最后再在Obsidian中安装上Image Auto Upload插件，配置一下就可以起飞使用。
