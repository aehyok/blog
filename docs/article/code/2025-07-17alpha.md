## gas gasUsed gasPrice 作用
  ```
  gas

  作用：交易的gas限制（gas limit）
  含义：这笔交易最多可以消耗的gas数量
  设置：由发送方在提交交易时预先设定
  目的：防止交易消耗过多资源，避免无限循环等问题

  gasUsed

  作用：交易实际消耗的gas数量
  含义：这笔交易在执行过程中真正使用了多少gas
  特点：总是 ≤ gas（gas限制）
  计费基础：实际的交易费用按照gasUsed计算

  gasPrice

  作用：每单位gas的价格
  单位：通常以Gwei为单位（1 Gwei = 10^-9 ETH）
  影响：决定交易的优先级，价格越高越容易被矿工打包
  ```
