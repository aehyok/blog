## 前言

vim就是一款文本编辑器，跟window下的notepad类似，只不过


## 打开某个文件
```javascript
vim /etc/nginx/nginx.conf
```

如果想让文件显示行号,下面两个命令都可以
```
:set number
:set nu
```
