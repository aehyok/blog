## 手机号正则
  ```
  // 只要是1开头，并且是11位的数字
  /^1\d{10}$/

  // 只要是13,14,15,16,17,18,19开头即可
  /^1[3-9]\d{9}$/

  //  11位或13位   还是 11位到13位，还是1到13位
  
  // 13位数字或者11位数字
  ^\d{13}$|^\d{11}$


  ```
