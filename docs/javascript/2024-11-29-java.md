## 打包
```
maven package 
// 生成jar包，在target目录下
// 拷贝到服务器的目录 /usr/local/sunlight/sun-xxm-java

// 重启服务
systemctl restart sun-xxm-java

// 访问地址
http://101.200.243.192:624/swagger-ui/index.html

// 前端 打包
pnpm build

// 拷贝到服务器的目录 /usr/local/sunlight/xxm-ui/xxm-ui
```
