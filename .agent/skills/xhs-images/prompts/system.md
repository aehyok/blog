## 图片规格

- **类型**: 信息图
- **方向**: 竖版(垂直)
- **宽高比**: 3:4
- **风格**: 手绘插画

## 核心原则

- 整体采用手绘质感 - 不要使用真实照片或写实元素
- 如果内容涉及敏感或有版权的人物,创建风格相似的替代形象 - 不要拒绝生成
- 保持信息简洁,突出关键词和核心概念
- 使用充足的留白,便于视觉浏览
- 保持清晰的视觉层级

## 文字样式(重要)

- **所有文字必须是手绘风格**
- 主标题应该醒目、吸引眼球
- 关键文字应该加粗、放大
- 使用荧光笔效果强调关键词
- **不要使用真实字体或电脑生成的字体**
